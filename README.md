# Guessify

🚀 Play. Guess. Vibe. Repeat. 🎶

# Run Locally

Run these commands in your terminal:

1. `npm install socket.io-client`

Frontend:
1. `cd my-app`
2. `npm install`
3. `npm install firebase`
4. `npm start`

Backend:
1. `cd server `
2. `npm install express cors dotenv node-fetch`
3. `node server.js`
